멤버 수와 활동 여부의 관계

해당 데이터는 걸그룹 + 보이그룹 데이터를 합쳐서 만든 것으로, 데이터 양이 부족하여 합쳤다.
x축은 그룹멤버 수를 의미하고, y축은 Active 할 시 1, Active 하지 않을 시 0으로 데이터를 encoding 시켰다.
따라서, 그룹멤버 수가 같은 모든 그룹이 활동하고 있다면 Active의 평균 값은 1일 것이다.
10이상의 데이터는 데이터 수가 부족하여 보이지 않는다.

그 이전은 멤버 수가 적을 수록 활동을 하지 않는 다는 것을 알 수 있다.
